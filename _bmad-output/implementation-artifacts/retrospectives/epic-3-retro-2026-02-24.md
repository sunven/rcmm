## 成功之处

### 1. 技术方案优化
- FIFinderSyncController API 替代 pluginkit CLI（更可靠，无需解析 CLI 输出）
- NSWindow + NSHostingView 替代 SwiftUI Window scene（MenuBarExtra app 中更可靠）
- SMAppService 登录项管理（macOS 13+ 原生支持）

### 2. 3 步引导完整
- 步骤 1：启用扩展（自动轮询 + 手动检测）
- 步骤 2：选择应用（扫描 + 预选常见工具 + 手动添加）
- 步骤 3：验证完成（操作指引 + 开机自启选项）

### 3. 用户体验设计
- 扩展已启用时自动跳过步骤 1
- 预选常见开发工具（Terminal、VS Code、iTerm2）
- 每步都支持"跳过"，不强制用户

### 4. 无障碍支持
- 所有交互元素有 accessibilityLabel
- 步骤指示器读出当前状态
- 键盘 Tab 导航可用

### 5. 复用 Epic 1-2 模式
- AppDiscoveryService 复用
- saveAndSync 配置同步模式复用
- 并发模式复用，无新的安全/并发问题

## 挑战与改进

### 挑战

1. **MenuBarExtra app 中 Window scene 不可靠**
   - SwiftUI Window scene 无法在启动时自动弹出
   - 解决方案：使用 NSWindow + NSHostingView

2. **功能遗漏**
   - 完成确认视图缺失
   - SMAppService unregister 分支缺失
   - 空状态缺少手动添加按钮

3. **macOS 版本兼容性**
   - macOS 15.0-15.1 系统设置 Extension UI 缺失
   - macOS 26 ARM FinderSync bug

### 改进措施

1. 功能验收时对照 AC 逐条验证
2. 记录 macOS 版本兼容性注意事项
