## 成功之处

### 1. 应用发现能力完善
- 扫描 /Applications 和 ~/Applications
- 按类型分组（终端/编辑器/其他）
- 支持手动添加任意 .app
- 14 个新单元测试覆盖

### 2. 设置窗口完整
- TabView 3 Tab（菜单配置/通用/关于）
- 菜单配置列表 + 增删功能
- 应用选择 Sheet（分组展示 + 勾选添加）
- 完整无障碍支持（VoiceOver + 键盘导航）

### 3. 拖拽排序优雅
- SwiftUI List + .onMove 原生支持
- 默认项视觉标识（star.fill + "默认"）
- VoiceOver 辅助操作（上移/下移）

### 4. 实时同步验证
- Darwin 通知 + App Group UserDefaults
- 配置变更 ≤ 1 秒生效
- 通用 `open -a` 命令支持多数应用

### 5. Code Review 持续有效
- 每个 Story 都有 Review 修复记录
- 安全性和并发问题在 Review 阶段被发现并修复

## 挑战与改进

### 挑战

1. **并发和线程安全问题反复出现**
   - Sendable 合规问题
   - 竞态条件（syncScriptsInBackground）
   - use-after-free（DarwinObservation.cancel）

2. **功能遗漏**
   - 批量添加触发 N 次 saveAndSync
   - 手动添加缺少重复检查
   - 状态信息未实现

3. **依赖边界复杂**
   - RCMMShared 禁止依赖 AppKit
   - AppDiscoveryService 必须放在 RCMMApp
   - AppInfo.icon 通过 extension 提供

### 改进措施

1. 创建并发安全检查清单供后续 Story 参考
2. 考虑创建 RCMMApp 测试目标
